## Issues Encountered

After installing Cursor, I was initially launched into the Agent Window interface. The layout was significantly different from the tutorials and documentation I found online, which made it difficult to locate the Extensions marketplace. I also tried using AI assistance to find the correct navigation path, but the instructions did not match the interface I was seeing.

After comparing the interface with screenshots and videos, I realized that I was in Cursor's Agent Window rather than the Editor Window. I explored the application further and found the Editor button in the top-right corner, which opened the traditional VS Code-style interface.

### Resolution

Once I switched to the Editor Window, I was able to access the Extensions marketplace, install both the Claude Code and Codex extensions, and complete the remaining setup steps successfully.
